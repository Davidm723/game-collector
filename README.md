# The Game Collector

This project was inspired by my own personal game collection. I wanted a place where you can manage your personal game collection in a fun and interesting way.

[The Game Collector](https://game-collector-y0n7.onrender.com/)

## Technologies Used

* Python
* HTML
* CSS
* Grid
* dj-database-url
* Django
* gunicorn
* psycopg2-binary
* python-doten
* whitenoise
